📡 POCSAG 2025 - DokumentationPOCSAG 2025 är ett Python-baserat system för att avkoda och hantera POCSAG-meddelanden med hjälp av en RTL-SDR.Systemet erbjuder ett webbaserat användargränssnitt för övervakning, filtrering och e-postnotifieringar av mottagna meddelanden.Utvecklad av: SA7BNB - Anders Isaksson✨ Nyheter i version 2025🆕 Modulär arkitektur för bättre stabilitet.🆕 Blacklist-funktion med realtidsuppdatering.🆕 Förbättrad e-posthantering med stöd för flera mottagare och testfunktion.🆕 Säkerhetssystem med autentisering, sessioner och brute force-skydd.🆕 Moderniserat webbgränssnitt med responsiv design.🆕 Docker-stöd för enkel installation.🚀 Installation (Raspberry Pi 4)1. Förbered operativsystemetAnvänd Raspberry Pi Imager för att installera OS:Enhet: Raspberry Pi 4.OS: Raspberry Pi OS Lite (64-bit).Inställningar (Kugghjulet): * Aktivera SSH.Ställ in användarnamn och lösenord.Konfigurera Wi-Fi.Skriv till SD-kort eller USB-minne.2. Hårdvara & UppstartAnslut din RTL-SDR v4 till en av USB-portarna på din Pi.Starta din Pi och hitta dess IP-adress (via routern eller ping raspberrypi.local).3. Installation via SSHAnslut via terminalen:Bashssh användarnamn@DIN-IP-ADRESS
Kör följande kommando för att installera systemet:Bashcurl -fsSL https://raw.githubusercontent.com/sa7bnb/pocsag2025/main/install.sh | bash
🔑 Kom igångFörsta inloggningNär installationen är klar når du systemet via din webbläsare:Surfa till: http://<pi-ipadress>:5000/Klicka på länken "Sätt upp ditt konto här".Skapa ditt administratörskonto (detta krävs vid första körningen för att låsa systemet).⚡ KonfigurationDu kan sköta all konfiguration direkt via webbgränssnittet:Frekvens: Ställ in önskad frekvens i MHz (t.ex. 161.4375).RIC-filter: Lägg till de numeriska adresser du vill bevaka.Blacklist: Blockera specifika adresser eller ord. Kan hanteras via webben eller direkt i config.json.E-post: Konfigurera SMTP-inställningar, mottagarlistor och anpassade ämnesrader för notifieringar.🛠 Huvudfunktioner📡 Radiomottagning & MeddelandenStöder POCSAG512 och POCSAG1200.Realtidsvisning av mottagna meddelanden med stöd för svenska tecken (åäö).Automatisk generering av kartlänkar om meddelandet innehåller koordinater.🔐 SäkerhetBCrypt-hashning av lösenord.Sessioner med timeout och brute force-skydd för inloggning.📂 ArkitekturProjektet är uppdelat i flera moduler:config_manager.py – Hanterar inställningar och dataklasser.email_handler.py – Sköter utskick av e-post.message_handler.py – Logik för avkodning och filtrering.server.py – Huvudserver och webbgränssnitt (Flask).💻 SystemkravKomponentKravHårdvaraRaspberry Pi 4, RTL-SDR v4, AntennOSRaspberry Pi OS Lite (64-bit)ProgramvaraPython 3.7+, multimon-ng, rtl-sdrUtvecklad av Anders Isaksson (SA7BNB).
